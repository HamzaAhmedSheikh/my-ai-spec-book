# Module 3: The AI-Robot Brain (NVIDIA Isaac)

## Overview

This module focuses on advanced perception and training using NVIDIA Isaac platform. You'll learn to leverage photorealistic simulation and hardware-accelerated robotics tools.

This module will guide you through:

1. **Chapter 11**: Introduction to NVIDIA Isaac - Platform overview
2. **Chapter 12**: Isaac Sim - Photorealistic simulation
3. **Chapter 13**: Synthetic Data Generation - Training data creation
4. **Chapter 14**: Isaac ROS - Hardware-accelerated VSLAM and navigation
5. **Chapter 15**: Nav2 for Bipedal Humanoids - Path planning for humanoid robots

---

## Learning Objectives

By the end of this module, you will be able to:

- Use NVIDIA Isaac Sim for photorealistic robot simulation
- Generate synthetic training data for AI models
- Implement hardware-accelerated VSLAM
- Configure Nav2 for humanoid robot navigation
- Deploy AI models on physical robots

---

## Prerequisites

- Completion of Module 1 (ROS 2)
- Completion of Module 2 (Gazebo & Unity)
- Basic understanding of computer vision
- NVIDIA GPU access (recommended)
